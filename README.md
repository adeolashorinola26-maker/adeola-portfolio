# Adeola Portfolio

A modern personal developer portfolio built with HTML, CSS, and JavaScript, showcasing frontend design, full-stack ambition, selected project concepts, and contact features.

## Live Site
[View Portfolio](https://adeolashorinola26-maker.github.io/adeola-portfolio/)

## Overview
This project is my personal portfolio website, designed to present my growing journey into web development in a clean, modern, and professional way. It highlights my interest in frontend development, backend thinking, responsive layouts, and polished user experience.

## Features
- Modern responsive design
- Hero section with personal branding
- About section
- Services/focus section
- Project showcase section
- Technology stack section
- Contact section
- Video-ready showcase area
- Smooth layout and styling with CSS
- Interactive elements using JavaScript
